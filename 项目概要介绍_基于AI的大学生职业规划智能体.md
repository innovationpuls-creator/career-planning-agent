# 基于AI的大学生职业规划智能体 — 项目概要介绍

## Context

本项目"基于AI的大学生职业规划智能体"是为中国大学生服务外包创新创业大赛（A类企业命题）开发的参赛作品。命题方为明杉科技，要求构建一个能够帮助大学生解决职业规划难题的AI智能体系统。

**核心问题：**
- 大学生自我认知模糊，定位偏差
- 职业信息不对称，认知片面
- 外部支持体系薄弱，指导缺位
- 规划落地性差，缺乏实践

**已实现的系统功能完全覆盖命题要求：**
- ✅ 就业岗位要求画像（12维度岗位画像 + 垂直图谱 + 换岗路径图谱）
- ✅ 学生就业能力画像（简历解析 + 12维度能力提取 + 竞争力评分）
- ✅ 职业生涯发展报告（人岗匹配 + 路径规划 + 行动计划 + 导出）

# 第一部分：项目概要介绍

## 一、项目概述

### 1.1 项目名称
**基于AI的大学生职业规划智能体**（Career Planning AI Agent）

### 1.2 命题来源
- **赛事**：中国大学生服务外包创新创业大赛
- **类别**：A类企业命题
- **命题企业**：明杉科技
- **方向**：智能计算 · 应用类

### 1.3 项目背景

在数字经济与产业升级双轮驱动的时代背景下，大学生就业问题已从单一的人才输送命题，演变为检验教育适配性、技术前瞻性和社会协同性的复杂系统工程。当前，我国就业市场呈现出"冰火两重天"的显著特征：一方面，智能制造、量子信息等战略新兴产业面临大量的人才缺口；另一方面，传统专业毕业生待业率长期徘徊在23%以上，结构性失业问题凸显。

当代大学生在职业规划中面临四大核心痛点：

| 痛点 | 具体表现 |
|------|---------|
| **自我认知模糊** | 专业选择受分数、家长意愿等因素影响，缺乏对自身兴趣、能力、性格的深度剖析；容易陷入"从众规划"误区 |
| **职业信息不对称** | 对行业和岗位的了解多来自社交媒体碎片化分享，对AI、新能源等新兴领域认知不足 |
| **外部支持薄弱** | 高校生涯规划课程重简历面试技巧，专业师资匮乏，家庭要么过度干预要么完全缺位 |
| **规划落地性差** | 职业规划停留在"纸面"，缺乏动态调整规划的能力，遭遇挫折易全盘否定原有规划 |

### 1.4 项目目标

本项目旨在构建一个**全流程、智能化、可解释**的大学生职业规划智能体，帮助学生：

- 快速了解就业市场对人才的能力要求
- 清晰准确分析自身的就业能力与意愿
- 获取明确可操作的职业生涯发展报告与行动计划

### 1.5 技术选型

| 层次 | 技术选型 | 说明 |
|------|---------|------|
| 后端框架 | FastAPI + Uvicorn | 高性能异步Python Web框架 |
| 数据库 | SQLite + SQLAlchemy | 关系型数据存储，支持PostgreSQL迁移 |
| 向量数据库 | Qdrant (本地) | 职位画像与学生画像的向量相似度检索 |
| 知识图谱 | Neo4j | 岗位要求画像的知识图谱可视化 |
| 大模型 | OpenAI兼容API (GPT-4o等) | 岗位画像提取、报告生成、能力分析 |
| 工作流引擎 | Dify | 简历解析对话、职业生涯目标规划 |
| 前端框架 | React + UmiJS (Alita) | 企业级React应用框架 |
| UI组件库 | Ant Design Pro v5 | 企业级UI组件 |
| 图表/图谱 | @antv/g6 + @ant-design/charts | 能力雷达图、知识图谱可视化 |
| 文档导出 | python-docx + reportlab | Word/PDF报告导出 |

### 3.2 模块一：就业岗位要求画像

#### 3.2.1 12维度岗位画像

系统从原始岗位数据中提取**12维度能力要求**，分为三组：

**专业与门槛组 (5维)：**
| 维度 | 名称 | 说明 |
|------|------|------|
| professional_skills | 专业技能 | 技能栈、工具、技术框架 |
| professional_background | 专业背景 | 专业/学科要求 |
| education_requirement | 学历要求 | 博士/硕士/本科/专科 |
| work_experience | 工作经验 | 年限、实习、项目经验 |
| other_special | 补充要求 | 证书、语言、证书资质 |

**协作与适应组 (3维)：**
| 维度 | 名称 | 说明 |
|------|------|------|
| teamwork | 团队协作 | 团队合作、配合要求 |
| stress_adaptability | 抗压适应 | 抗压能力、加班、出差 |
| communication | 沟通表达 | 沟通协调能力 |

**成长与素养组 (4维)：**
| 维度 | 名称 | 说明 |
|------|------|------|
| documentation_awareness | 文档规范 | 文档报告意识 |
| responsibility | 责任心 | 责任心、积极主动性 |
| learning_ability | 学习能力 | 学习能力、自学能力 |
| problem_solving | 问题解决 | 问题分析与解决能力 |

#### 3.2.2 垂直岗位图谱

展示同一岗位在不同行业/企业发展阶段的晋升路径：

| 级别 | 经验要求 | 典型特征 |
|------|---------|---------|
| **低级岗位** | 应届生/1-3年 | 基础技能、执行任务 |
| **中级岗位** | 3-5年 | 团队协作、独立项目 |
| **高级岗位** | 5年+ | 管理/专家路线 |

每个层级展示：薪资范围、典型公司、能力要求。

#### 3.2.3 换岗路径图谱

将相关岗位进行血缘关系关联，规划岗位转换路径：

- 系统构建了**5个以上**核心换岗路径
- 每条路径**≥2条**具体转换路线
- 基于向量相似度识别技能重叠度高的相邻岗位

### 3.3 模块二：学生就业能力画像

#### 3.3.1 简历解析

- 支持多种格式上传（图片、PDF、Word）
- 通过Dify工作流进行多模态解析
- 流式对话交互，支持补充信息

#### 3.3.2 12维度能力提取

采用与岗位画像一致的12维度框架提取学生能力：

- **完整度评分**：12维度填写完整度
- **竞争力评分**：与同龄人/目标岗位群体的对比

#### 3.3.3 能力缺口分析

自动对比学生能力画像与目标岗位要求，识别：

| 类型 | 说明 |
|------|------|
| **优势维度** | 超出岗位要求的能力 |
| **缺口维度** | 低于岗位要求的能力 |
| **匹配维度** | 基本满足要求的能力 |

### 3.4 模块三：职业生涯发展报告

#### 3.4.1 职业探索与岗位匹配

量化人岗匹配度，从四个维度分析：

| 维度 | 包含能力 | 默认权重 |
|------|---------|---------|
| **基础要求** | 学历、专业、工作经验 | 25% |
| **职业技能** | 专业技术栈 | 30% |
| **职业素养** | 沟通、抗压、协作 | 25% |
| **发展潜力** | 学习能力、问题解决 | 20% |

综合打分 = Σ(各维度匹配分 × 权重)

#### 3.4.2 职业目标设定与路径规划

- 结合职业探索结果与个人意愿制定职业目标
- 分析目标职业的社会需求与行业发展趋势
- 构建清晰的发展路径（例：UI设计师 → 资深体验设计组长 → 设计总监）

#### 3.4.3 行动计划与成果展示

分阶段个性化成长计划：

| 阶段 | 时间范围 | 核心理念 |
|------|---------|---------|
| **短期计划** | 0-3个月 | 技能入门、基础项目 |
| **中期计划** | 3-9个月 | 项目实践、实习积累 |
| **长期计划** | 9-24个月 | 深化专业、管理能力 |

每个阶段包含：学习路径、实践安排、评估指标。

#### 3.4.4 报告编辑与导出

- AI智能润色、内容完整性检查
- 手动编辑调整
- 一键导出Word/PDF格式

### 3.5 模块四：管理与数据看板

#### 3.5.1 用户管理

#### 3.5.2 岗位数据管理
